# Rime custom template overrides

- Put your custom Rime patches here as `*.custom.yaml.tmpl`.
- These templates are synced directly by chezmoi into the active Rime user directory.
- Generic files like `default.custom.yaml.tmpl` and `rime_mint.custom.yaml.tmpl` can be wrapped for multiple frontends.
- `squirrel.custom.yaml.tmpl` is synced only on macOS Squirrel.
- `ibus_rime.custom.yaml.tmpl` is synced only on Linux `ibus-rime`.
